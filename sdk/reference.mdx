## `World` Class

The `World` class represents a single knowledge base instance.

### Constructor

```typescript
new World(options: {
  baseUrl?: string;
  apiKey: string;
  worldId: string;
  fetch?: typeof fetch;
})
```

### Methods

#### `sparql`

Executes a SPARQL query or update against the world.

```typescript
sparql(query: string, options?: { accountId?: string }): Promise<SparqlResults | null>
```

#### `search`

Performs a semantic search within the world.

```typescript
search(query: string, options?: {
  limit?: number;
  offset?: number;
  accountId?: string;
}): Promise<SearchResult>
```

#### `get`

Retrieves the world's metadata.

```typescript
get(options?: { accountId?: string }): Promise<WorldRecord | null>
```

#### `update`

Updates the world's metadata.

```typescript
update(data: WorldRecord, options?: { accountId?: string }): Promise<void>
```

#### `remove`

Deletes the world.

```typescript
remove(options?: { accountId?: string }): Promise<void>
```
